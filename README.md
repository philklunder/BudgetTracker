# Budget Tracker (WPF, C#)

A simple desktop application built to learn C# and WPF.

## Features

- Add income and expenses
- View a list of all transactions
- Automatically calculate the current balance

## Technologies

- C#
- WPF (.NET)

## Purpose

This project is created as a learning exercise to improve my understanding of object-oriented programming and desktop application development using C#.

## Planned Improvements

- Edit and delete transactions
- Save and load data using JSON
- Filter transactions by category
- Monthly overview and basic analytics
- Refactor the project using the MVVM pattern
